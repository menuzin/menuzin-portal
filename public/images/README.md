# Images Directory

Place the following images in this directory for the Menufy landing page:

- `hero-food.jpg` - Hero section food image (square/aspect-square recommended)
- `guest-food.jpg` - Guest experience section food image (vertical/aspect-4/5 recommended)
- `menu-ui.png` - Mobile menu UI preview (vertical/aspect-9/16 recommended)
- `dashboard.png` - Owner portal/dashboard interface (landscape/aspect-video recommended)

If images are missing, the application will display graceful placeholder UI elements with icons.
